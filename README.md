**TUTORIAL PENGGUNAAN WEBSITE ALFA COMPUTER**

**TAMPILAN AWAL WEBSITE**
![1(halaman pertama index php)](https://github.com/Aminarti/project-akhir/assets/120153116/bc0ad493-bbfa-4655-a043-98fab8e8a5f3)

Dapat dilihat pada gambar tampilan awal website berisikan tentang brand apa saja yang terdapat Alfa Computer.
Kemudian untuk masuk ke website pengguna dapat mengklik *“Login”* yang berada di ujung kanan website.
Terdapat 3 role dalam website ini yaitu :
1. User
2. Admin 
3. Karyawan


```USER```

**USER = Login**
![2(login user)](https://github.com/Aminarti/project-akhir/assets/120153116/31c35766-ae57-4c30-bcf0-1d72e5dda7a8)

Apabila user ingin login kedalam web user harus memiliki akun dan menginputkan username dan password yang telah dimiliki lalu klik *“Sign In”*. Apabila berhasil login maka user akan masuk ke halaman utama website sebagai user. 

Namun apabila belum memiliki akun pengguna dapat membuat akun terlebih dahulu dengan cara mengklik *“Daftar Sekarang!”* di bawah tombol Sign In.


**USER = Registrasi / Sign Up**
![3(registrasi user)](https://github.com/Aminarti/project-akhir/assets/120153116/73118756-26c2-44f6-895e-bf0787b081eb)

Registrasi / Sign Up dilakukan apabila user belum memiliki akun. Pertama user menginputkan Username, lalu Password dan mengetik kembali Password tersebut. Lalu klik *“Submit”* apabila berhasil membuat akun, maka dapat login menggunakan username dan password yang tadi telah dibuat.

**TAMPILAN WEBSITE (Role = USER)**
![4(dashboard user)](https://github.com/Aminarti/project-akhir/assets/120153116/68e76b54-8aa5-409c-9e6e-cac5044d934e)

Tampilan awal pada website memperkenalkan sedikit tentang Alfa Computer. Pada bagian atas website terdapat halaman Home, Products,Transaksi, About Us, Contact, User Profile dan Logout. 
1. PRODUCTS : Apabila ingin melihat produk yang ditawarkan user dapat mengklik *“Products”* maka website akan menampilkan produk apa saja yang ditawarkan di Alfa Computer.
2. TRANSAKSI : Apabila user ingin melihat transaksi maka dapat mengklik *“Transaksi”* kemudian akan menampilkan riwayat tranksaksi yang telah dilakukan user.
3. ABOUT US : Apabila user ingin menampilkan tentang toko alfa computer seperti visi dan misi lalu alat toko terdapat pada about us dapat mengklik *“About Us”*.
4. CONTACT : Apabila user ingin menghubungi alfa computer dapat mengklik *"Contact"* yang akan otomatis akan terhubung ke WhatsApp.
5. USER PROFILE : Apabila user ingin melihat profile pengguna dapat mengklik *"User Profil"*.
6. LOGOUt : Kemudian apabila user ingin keluar dari website dapat mengklik *“Logout”* yang terdapat pada ujung kanan website.

**USER = Melihat produk dan pesan barang**
![5(menu products user)](https://github.com/Aminarti/project-akhir/assets/120153116/4ef346f6-1b21-4029-a436-c5d2bb4225fc)

Untuk melihat produk dan melakukan pemesanan barang user dapat mengklik *“Products”*. Lalu pada website akan menampikan produk apa saja yang ditawarkan oleh Alfa Computer. Berisikan Nama Barang, Harga, Stok dan Gambar Barang.

**USER= Pesan barang**
![6(user mau pesan barang)](https://github.com/Aminarti/project-akhir/assets/120153116/8f132886-7acf-45ce-bb51-ed4486bfcde9)



<img width="536" alt="7(user berhasil pesan) (1)" src="https://github.com/Aminarti/project-akhir/assets/120153116/37db7bf0-60f1-41e6-beea-28b38a2d4642">

Apabila user ingin memesan barang dapat dilakukan dengan langkah langkah sebagai berikut : 
- Pertama apabila user ingin memesan barang dapat mengisi form *“Pesan Barang”*. 
- User mengisi mulai dari ID Barang, Jumlah Barang dan Alamat. 
- Kemudian untuk Total Harga akan muncul otomatis sesuai dengan jumlah barang yang dipesan. 
- Pastikan semua data yang di input telah benar, lalu  klik *“Pesan”*.
- Apabila user berhasil memesan barang maka akan muncul notifikasi barang berhasil dipesan. Untuk detail transaksi dapat dilihat pada menu transaksi.

**USER = Melihat riwayat transaksi**
![8(riwayat transaksi user)](https://github.com/Aminarti/project-akhir/assets/120153116/06308dfd-58a5-4ce7-8b9d-a4c545496303)

Klik *“Transaksi”* untuk melihat riwayat transaksi yang telah dilakukan oleh user. Riwayat Transaksi ini berisikan riwayat pemesanan barang yang dilakukan user seperti nama barang, jumlah barang yang dipesan, tanggal transaksi dan juga status pengiriman barang apakah sudah dikirim atau belum.
Isi dari Data Transaksi yaitu :
- ID Transaksi
- User
- ID Barang
- Nama Barang
- Tanggal Transaksi
- Alamat
- Total Bayar
- Status

**USER = Melihat about us**
![9(about us bagian user)](https://github.com/Aminarti/project-akhir/assets/120153116/887e959f-a1f1-46a2-9614-33032dad77f4)

Klik *“About Us”* untuk apabila user ingin melihat tentang toko Alfa Computer, tentang visi misi dan untuk melihat alamat toko.

**USER = Menampilkan menu profile pengguna**
![10(menu profile bagian user)](https://github.com/Aminarti/project-akhir/assets/120153116/1323cd18-c0e2-40b5-b533-532adfa125df)
Klik *“User Profile”* apabila user ingin melihat profil pengguna.
Isi data dari user profile yaitu :
- Nama Lengkap
- Email
- Nomor HP
- Alamat
Apabila ingin mengubah profile dapat langsung mengubah isi data tersebut kemudian mengklik “simpan perubahan” maka perubahan akan berhasil disimpan.

- Kemudian website akan menampilkan form untuk pembuatan akun karyawan. 
- Selanjutnya admin menginput Username, Password dan ketik kembali Password lalu klik *“Buat”* , maka akun karyawan berhasil dibuat.

Apabila karyawan ingin login kedalam web karyawan harus memiliki akun dan menginputkan username dan password yang telah dimiliki lalu klik *“Sign In”*. **TAMPILAN WEBSITE (Role = KARYAWAN)**
![12(crud karyawan)](https://github.com/Aminarti/project-akhir/assets/120153116/f12d9dd8-07b3-4126-9483-57f29619c293)

Karyawan Alfa Computer bertugas mengelola transaksi barang  yaitu melakukan edit barang seperti :
1. Mengupdate transaksi
2. Menghapus barang 
3. Mengupdate barang 
4. Menambah barang

**KARYAWAN = Update Transaksi**
![13(update transaksi bagian karaywan)](https://github.com/Aminarti/project-akhir/assets/120153116/a1547936-aa48-4f93-a0c9-e7d0801687d4)

![14(sweet alert karywan berhasil update)](https://github.com/Aminarti/project-akhir/assets/120153116/d080d9b7-8e11-4ce7-98da-9a12281bf70a)
Apabila karyawan ingin mengupdate status pengiriman barang dapat dilakukan dengan langkah langkah sebagai berikut : 
- Pertama karyawan mengklik *“Update”* pada halaman utama.
- Kemudian Website akan menampilkan riwayat transaksi pemesanan barang.
- Lalu untuk mengupdate status barang dilakukan dengan mengklik *“Kirim Barang”* maka status barang akan otomatis berubah.
- Setelah itu akan muncul pop-up notifikasi bahwa berhasil mengupdate data transaksi.

**KARYAWAN = Tambah Data Barang**
![15(menu tambah data karywan)](https://github.com/Aminarti/project-akhir/assets/120153116/0bfe3014-0408-451f-ba89-771a625b528e)

Apabila karyawan ingin menambahkan data barang dapat dilakukan dengan langkah langkah sebagai berikut : 
- Pertama karyawan mengklik *“Tambah”* pada halaman utama.
- Kemudian website akan menampilkan form untuk menambahkan data barang.
- Karyawan mengisi mulai dari ID Barang, Nama Barang, Harga, Stok dan memasukkan gambar barang.
- Klik *“Tambah”* untuk menyimpan data barang baru. Maka Data barang baru berhasil ditambahkan.

**KARYAWAN = Update Data Barang**
![16(menu update data barang karyawan) (2)](https://github.com/Aminarti/project-akhir/assets/120153116/2df212e8-af40-4eaa-9d79-c36d01b0e45d)

Apabila karyawan ingin melakukan update data barang dapat dilakukan dengan langkah langkah sebagai berikut : 
- Pertama karyawan mengklik *“Update”* pada halaman utama.
- Kemudian website akan menampilkan form untuk update data barang.
- Karyawan mengisi data yang diubah seperti ID Barang, Nama Barang, Harga atau Stok.
- Klik *“Update”* untuk menyimpan update data barang. Maka data barang berhasil diubah.
